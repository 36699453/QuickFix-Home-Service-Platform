# 🛠 QuickFix - Smart Home Service Platform

## 📖 Overview

QuickFix is a professional ASP.NET Core MVC web application that allows customers to book trusted home services online.

The project demonstrates enterprise-level backend development using ASP.NET Core, Entity Framework Core, SQL Server, Repository Pattern, Dependency Injection, REST APIs, and Swagger.

---

# 🚀 Features

- User Registration
- User Login
- Session Authentication
- Home Service Booking
- Admin Dashboard
- Service Management (CRUD)
- Booking Management
- REST API
- Swagger Documentation
